# SafeHer 🌸
### Women SOS Safety iOS App

A personal safety iOS app designed to empower women by giving them a fast, reliable, and discreet way to call for help in dangerous situations.

## Features
- 🔴 SOS Button — Hold 3 seconds to alert emergency contacts
- 📍 Live GPS Location — Sends Google Maps link via WhatsApp
- 📞 Fake Call — Simulate an incoming call to exit unsafe situations
- 🔔 Siren Alert — Loud siren and vibration on SOS trigger
- 👥 Trusted Contacts — Manage up to 5 emergency contacts

## Tech Stack
- Swift + SwiftUI
- CoreLocation (GPS)
- AudioToolbox (Siren)
- Firebase Auth + Realtime DB
- WhatsApp URL Scheme
